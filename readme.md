# Nimbus Perfect CJK

A powerful, multilingual typeface project fully built via **AI + Python** automation scripts. It seamlessly merges the classic Western font **Nimbus Roman** with the core multi-regional variants (SC/TC/JP/KR) of **Source Han Serif / Noto Serif**, completely eliminating missing characters, broken glyphs, and visual mismatches in multilingual layouts.

Don't let its elegant Western appearance fool you—under the hood lies a true **Unicode Leviathan**!

### Key Features

* **Seamless Multi-Regional Fusion**: Utilizes Python automation to flawlessly merge and optimize font weights and glyph streams across all CJK regions.

* **Universal CJK Support**: Delivers gorgeous Western typography while fully supporting **Simplified Chinese, Traditional Chinese, Japanese (Hiragana/Katakana/Kanji), and Korean (Hangul)**. One font package to cover them all.

* **Modern Development Pipeline**: Moves away from tedious manual glyph stitching. Built with AI collaboration + Python scripts to directly extract, rewrite, and encapsulate low-level font metrics with absolute precision.

### Built With

* **Python**: Powered by `fontTools` for advanced font merging, GSUB table reconstruction, and Unicode remapping.

* **Google Search AI Mode**: Assisted in data conflict resolution and optimizing the build pipeline logic.
